# Projeto Aerocode

## 

Este projeto tem como objetivo desenvolver uma aplicação utilizando TypeScript, focando no desenvolvimento de uma aplicação que simula o desenvolvimento de uma aeronave. O sistema conta com cadastro de funcionários, criação de aeronaves, adição de peças, etapas e testes necessários.

---

## Execução do Projeto


### Instalação de dependências

```bash
npm install
```

---

### Compilação do TypeScript

```bash
tsc
```

---

### Execução do projeto

## Reset de informações

É possível fazer um reset dos dados criados e restaurar o projeto para apenas 3 funcionários criados e nenhuma aeronave, executando o comando:

```bash
node out/cli/index.js
```

## Inicialização do projeto

Da mesma maneira, para iniciar o projeto normalmente e acessar o menu de opções, basta executar:

```bash
node out/cli/menu.js
```
